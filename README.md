# OpenCV Image Viewer and Grayscale Converter

This is my first OpenCV mini project built using Python.

## Features
- Reads an image from the system
- Checks if the image is loaded successfully
- Converts a color image (BGR) to grayscale
- Allows user to:
  - View the grayscale image
  - Save the grayscale image

## Technologies Used
- Python
- OpenCV

## How to Run
```bash
python A1.py
